This first setup can be used without having access to the app source code or setting any special debug settings in the SDK itself.Download a traffic monitoring tool: Wireshark Watch the Webtrends requests being made from your app as they happen.Download Wireshark.  Wireshark is a free open source network request and packet analyzer (Wikipedia).  Install it onto a machine where you can run an emulator for the app or plug your device(s) into and make your internet requests through.
