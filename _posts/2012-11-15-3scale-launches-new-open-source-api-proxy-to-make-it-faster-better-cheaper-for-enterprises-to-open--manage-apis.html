SAN FRANCISCO, CA – 3scale (www.3scale.net), the fastest growing provider of SaaS API Management solutions and services, today announced the launch of a new open source API proxy providing enterprises with high performance on premise and cloud API traffic management.Built on the highly acclaimed NGINX Web Server technology, the new 3scale open source API Proxy combined with 3scale’s out-of-the box API Management solution satisfies the pressing business and technical needs of API providers to: Easily (and securely) open and manage APIs, Launch APIs with the fastest time-to-market, Keep control on their API architecture and, Use proven technologies in the most demanding production environments.“With the new 3scale open source API proxy it has never been easier for companies to open and manage APIs.
