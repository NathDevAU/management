The open-source management layer for APIs, delivering high performance and reliability.Built on top of NGINX, Kong can be fully operated with a simple and easy to use RESTful API.Put your services behind Kong and add powerful functionality through Kong Plugins, in one command.Kong can run anywhere, in the cloud or on-premise - in a single, hybrid or multi-datacenter setup.Kong allows developers to reduce complexity and deployment times in implementing an API solution on the NGINX platform.
