Akana, a Microsoft Gold Certified Partner and an API Management and Cloud Integration leader, announced today that it has earned the new Microsoft Cloud & Enterprise Software Certification.The certification covers Akana’s family of API management products for use on Windows Server 2012 and the Microsoft platform.As a member of the Microsoft Partner Network for 10 years, Akana offers full lifecycle management of Microsoft-based APIs and services.Akana’s API management tools help businesses to share data as APIs, connect and integrate applications, drive developer adoption, and monitor and analyze their API usage.Enterprises can use the new solution on Microsoft Azure, on-premise Microsoft environments for internal services, and hybrid cloud scenarios that expose internal services securely to mobile devices or to the cloud.
