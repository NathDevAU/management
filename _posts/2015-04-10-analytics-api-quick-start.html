In this quick start you will perform the following tasks: Obtain credentials to use the Analytics API Generate reports on each of the Analytics API dimensions Generate an engagement report for the account For details on the API used in this document you can view the Analytics API Overview document and the Analytics API reference.The Analytics APIs security is implemented using OAuth2.To permit you to get started using the Analytics API as quickly as possible, the API calls in this quick start will use a shell script called api-tester.sh to get access tokens and make the API calls.The api-tester script uses cURL to make http calls.
